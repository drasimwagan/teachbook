How it works

Three panes, stepping forward together.

Every lesson is a sequence of steps. When you hit Next, the visualization advances, the solution code highlights the lines that ran, the narration updates, and the source view rails its current scene block — just like a debugger.

  • 1 Concept — rendered Markdown with LaTeX, or raw source for editing. Click any step to jump.
  • 2 Visualization — SVG scene that tweens between steps, plus a read-only code view with the active line marked.
  • 3 Chat — streaming conversation with Claude, aware of the current step and the full notebook.

Features

Everything a teaching app should do.

Built on top of your existing Claude Code subscription. Local. Fast. Extensible.

Streaming Claude chat

Token-level streaming via the claude CLI. Cancel with SIGTERM; partial tokens stay.

Generate whole notebooks

Describe a concept in a sentence. Claude streams a full .tbk with prose, code, visualizations, and a quiz.

Round-trip editing

On any step, click + Add step. Claude writes a new scene, validates, and splices it into the source. Undo/redo works across the board.

8 core primitives

grid · matrix · shape · arrow · label · axes · plot · graph — composable into algorithm traces, physics sims, DP tables, circuits.

Plugin system

Extend the engine with a new primitive for any domain. One TypeScript file, one registry line. Self-documents to Claude.

Local-first, open source

Runs on your machine; uses your Claude Code login; stores notebooks under ~/Teachbook. No accounts, no server, no telemetry. MIT.

One engine, every subject

15 notebooks ship out of the box.

Plugin system

Extend into any domain.

The core engine ships 8 general-purpose primitives. Domain-specific primitives — molecules, Bloch spheres, circuit schematics, neural networks — live as plugins. Each plugin is one TypeScript file; add it to the registry and it's available everywhere.

Plugins self-document their JSON shape into the format guide Claude sees, so Generate and + Add step produce valid plugin JSON on the first try.
